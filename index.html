Soporte para TPM 2.0 y Secure Boot: Reforzó las características de seguridad con el soporte para TPM (Trusted Platform Module) 2.0 y Secure Boot, lo que ayudó a proteger el arranque del sistema y los datos sensibles Interfaz de usuario: Combinó la interfaz clásica de escritorio con la funcionalidad de la pantalla de inicio de Windows 8, ofreciendo el menú de inicio mejorado con una mezcla de elementos tradicionales y tiles (baldosas). Cortana: Introdujo Cortana, el asistente virtual de Microsoft, para ayudar a los usuarios a realizar búsquedas, configurar recordatorios y otras tareas mediante comandos de voz. Microsoft Edge: Incluyó el navegador Microsoft Edge (luego reemplazado por la nueva versión basada en Chromium), ofreciendo una alternativa a Internet Explorer con mejoras en velocidad y compatibilidad. Escritorios virtuales: Implementó la funcionalidad de Escritorios Virtuales, permitiendo a los usuarios crear y gestionar múltiples escritorios para organizar mejor las tareas y aplicaciones abiertas. Centro de actividades: Introdujo un nuevo "Centro de actividades" (Action Center) que centralizó las notificaciones y las acciones rápidas para facilitar el acceso a configuraciones y alertas. Actualizaciones como servicio: Adoptó un modelo de "Actualizaciones como servicio," proporcionando actualizaciones de características y seguridad de manera más regular a través de Windows Update. Windows Ink: Introdujo Windows Ink, una suite de herramientas para tomar notas y hacer anotaciones en documentos y fotos utilizando dispositivos con pantalla táctil o lápices digitales. Mejoras en la seguridad: Incluyó mejoras en la seguridad, como el "Windows Defender" mejorado con protección en tiempo real y la integración con Windows Hello para autenticación biométrica.
Windows 11 11 10.0.22000 (10.0) 2021 Intel Core i3, i5, i7, i9 / AMD Ryzen 3, 5, 7, 9 2.90 GHz a 5 GHz (Dependiendo se su procesador y generacion) x86 (32 bits) y x64 (64 bits) 4 a 32 hilos (Depndiendo del tipo de procesador y su generacion) DDR4/DDR5 SDRAM 4 GB a 6 TB 2666 MHz-5600 MHz Requisitos mínimos de hardware: Windows 11 tiene requisitos de hardware más estrictos que Windows 10, incluyendo un procesador compatible de 64 bits, 4 GB de RAM, 64 GB de almacenamiento, TPM 2.0, y Secure Boot. Soporte para DirectX 12 Ultimate: Incluye soporte para DirectX 12 Ultimate, proporcionando una mayor capacidad para gráficos avanzados y juegos. Mejoras en la integración de hardware: Optimización para pantallas de alta resolución y nuevos estándares de conectividad, como USB-C y Thunderbolt. Requisitos para TPM 2.0 y Secure Boot: Aumenta los requisitos de seguridad con TPM 2.0 y Secure Boot como requisitos para la instalación del sistema operativo. Interfaz de usuario rediseñada: Introdujo un rediseño significativo en la interfaz de usuario con un menú de inicio centrado, nuevos íconos, y una barra de tareas mejorada. Windows Widgets: Añadió Windows Widgets, una nueva forma de acceder a información personalizada como noticias, clima, y calendario desde la barra de tareas. Mejoras en la multitarea: Introdujo Snap Layouts y Snap Groups para mejorar la gestión de ventanas abiertas y facilitar la organización del escritorio. Integración de Microsoft Teams: Integró Microsoft Teams directamente en la barra de tareas para facilitar las llamadas y la colaboración en línea. Mejoras en la administración de ventanas: Nuevas funcionalidades como "Desktops" mejoradas para organizar mejor las tareas y los proyectos en diferentes escritorios virtuales. Actualizaciones de rendimiento y eficiencia: Mejoras en el rendimiento general del sistema y en la eficiencia energética, especialmente para dispositivos portátiles y convertibles. Soporte para aplicaciones de Android: En colaboración con Amazon y la Microsoft Store, Windows 11 ofrece soporte para aplicaciones de Android a través de la Microsoft Store y la Amazon Appstore. Nuevas características de seguridad: Incluye mejoras en las herramientas de seguridad y administración, así como una mayor integración con características de seguridad avanzadas.
